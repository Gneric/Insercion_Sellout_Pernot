La primera vez se tiene que crear el enviroment con
python3 -m venv insertSelloutENV

Activar el enviroment
En windows
insertSelloutENV\Scripts\activate.bat

En Linux
source insertSelloutENV/bin/activate

luego instalamenos las librerias que estan en requerimientos con
pip install -r requerimientos.txt

Para ejecutar el programa ejecutar, colocar el archivo en /src/data
py main.py
